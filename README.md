# Trabalho de Lógica em Python

Projeto desenvolvido como trabalho acadêmico com foco em lógica de programação, estruturação de algoritmos e resolução de problemas utilizando Python.  
O projeto aplica conceitos fundamentais da programação para resolver situações propostas no exercício.

---

## Funcionalidades

- Implementação de algoritmos baseados em lógica  
- Entrada e saída de dados pelo terminal  
- Estruturação clara e organizada do código  
- Aplicação de estruturas condicionais e de repetição  
- Resolução prática de problemas propostos  

---

## Tecnologias Utilizadas

- Python — lógica e execução do programa
